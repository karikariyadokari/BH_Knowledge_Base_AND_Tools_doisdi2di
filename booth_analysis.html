📖 利用マニュアル

1. CSVの読み込み

右上「CSVファイル」からアップロード（複数可、ドラッグ&ドロップ対応）。

対応カラム: ROW, MAC, ENV, TYPE, TIME, NOTE, INFO

TIME は UTC → JST(+9h) 自動変換。2026-01-29 より前はテストデータとして除外。

集計TYPE: WIN_TURN / LOSE_TURN / DISCONNECTED_BY_APPLICATION / ERROR_MACHINE_NO_RESPONSE(Err:NR) / ERROR_MACHINE_REPORT(Err:MR) / ERROR(Err) / その他ERROR系(Err:他) / MAINTAIN_ON

2. プレイ料金設定

「🪙 プレイ料金設定」でゲーム内通貨のコストを設定。デフォルト料金は全ブース共通、ブース個別設定で上書き可。「適用して更新」で再計算されます。

3. ブース別詳細

ブース行をクリックで日付別テーブル展開。並び替えボタンで順序変更（同ボタン再クリックで昇順⇔降順）。ブース名・タグは直接クリックして編集。
